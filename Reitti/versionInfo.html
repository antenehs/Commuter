Commuter Version 3.0

Some of the many new features in this version include.

1. Usability

  • Commuter is now also available in Tampere region
  • Named bookmarks that can be easily created and managed. You'll find them very convenient once you start using them.
  • Long press anywhere on the map to save the location or go directly to it.
  • iOS maps app integration for searching public transit routes for a route opened in maps.
  • Automatic old history cleaning.
  • Search for routes from almost every where in the app.

2. User Interface

Commuter 3.0 has a totally redesigned UI for an even better experience. These include:

  • Support for iPhone 4,iPhone 5, iPhone 6/6 plus and iPad screen sizes.
  • Support for landscape screen orientation
  • Route search list has more information about route length and now also displays waiting times.
  • Newly designed bus stop annotations where you can get information about how long it takes to go there right from the home screen.

3. Performance improvements and bug fixes
